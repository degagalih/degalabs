UI Design: Mulai dari mana?

02 MAR 2021 | Galih

Hal pertama kali yang ditanyakan dalam melaksanakan suatu hal yang baru adalah harus memulainya dari mana?. Jawabannya adalah dengan membuat UI Design (Desain antarmuka), terdengar membingunkan dan cukup klise memang. Tapi itulah yang benar-benar harus dilakukan, bila saya mengutip salah satu slogan e-commerce terkenal di Indonesia “Mulai aja dulu”.

Apakah Benar Kamu Suka UI Design?. Jika ingin menjadi UI Designer, kita asumsikan bahwa kamu suka membuat UI Design dan mempunyai keterkaitan terhadap elemen-elemen yang membangunnya. Kamu juga suka mengamati UI Design yang tersebar di Internet.

Apabila kamu tidak suka membuat UI Design, maka tanyakan lagi kepada dirimu apa motivasi belajar UI Design?

Mulai Membuat UI Design Sendiri. Untuk mulai belajar UI Design, kamu tidak perlu mengetahui kaidah fundamental mengenai bagaimana suatu produk digital yang bagus, kamu juga tidak perlu belajar fundamental UI/UX. Kamu cukup memulainya dari membuat sebuah UI Design. Kamu bisa mencari referensinya di Internet sesuai dengan UI Design yang paling kamu sukai.

Kunci belajar UI Design menurut saya adalah fokus pada satu hal, kuasai satu hal tersebut, setelah itu baru mengembangkan ke hal lain.
